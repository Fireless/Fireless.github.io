Python学习笔记2:网页的获取

2019-03-01 Python

简介

Requests是唯一一个第三方的非转基因Http python库，本文主要利用Requests库对网页的HTML文件进行下载。

具体步骤

安装requests模块

安装指令为pip3 install requests,如无法安装则需要以管理员权限运行。

python3代码

#!/usr/bin/env python3
import requests

def download(url):

#检查URL是否存在
try:
    req = requests.get(url)
except requests.exceptions.MissingSchema:
    print('Invalid URL "{}"'.format(url))
    return
#检查是否成功访问了网站
if req.status_code == 403:
    print('You do not have the authority to this page')
    return

filename = url.split('/')[-1]
with open(filename,'w') as fobj:
    fobj.write(req.content.decode('utf-8'))
print("Download over.")

if __name__ == '__main__':
    url = input('Enter a URL: ')
    download(url)

运行后将在当前目录自动生成文件夹并进行下载。
