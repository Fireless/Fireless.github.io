Win10系统下vs code安装plantUML插件

2019-03-19 work

简介：因为环境变量的问题安装plantUML出现了一点问题，在此记录。

1.前置需求

  • Java
  • graphviz
  • vs code

2.安装java并为其配置环境变量

https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

从官网下载java，并根据提示步骤安装，安装完成后对环境变量进行配置。
我的电脑–>属性–>高级系统设置–>环境变量

编辑系统变量中的PATH条目，将本地java所在文件夹的路径添加进去，若加在开头注意已分号结束。

完成环境变量的配置后，打开win10自带的命令提示簿，检测java是否全局安装成功。

3.安装graphviz并配置环境变量

http://www.graphviz.org/download/

同理，安装完graphviz后需对其进行环境变量的配置，只需在PATH中添加graphviz的路径。经检测能显示版本号即可。

4.在vs code中安装plantUML插件

插件安装完成后，打开UML文件，按ALT + D 即可实时预览UML图
