Vue学习笔记（二）--引入vuetify

2019-04-02 vue

简介：

记录一些使用vue-cli构建项目后利用vuetify的组件编写页面时遇到的问题。

1.引入vuetify

在yarn add vuetify安装完vuetify，并且在main.js文件中引用vuetify后，vuetify组件任然无法正常显示。

只显示文本，并没有样式，所以猜测应该是没有成功引用。查阅官方文档后发现还需输入

生成vuetify.js文件之后才能正常使用vuetify。

2.根元素唯一

root element only one，当出现两个根元素时，代码会报错，无法正常运行。此时理解了vuetify要求所有component都需要用包裹的一个原因，这样做可以保证根元素的唯一性。

3.components内的问题

由于一开始发现根元素不唯一无法运行，所以所有component都用了。结果发现Home页面引用的组件只有一个能显示，其余组件失效。经过测试发现，如果Home页面引用的组件中有一个或多个标签，只会显示第一个。故components内Vue文件改用来保持其根元素的唯一性。在此之后发现页面能够正常显示。
