利用CSS中的span标签制作Google的logo

2019-03-03 CSS

主要为了熟悉span标签和HTML，CSS的应用。主要遇到三个问题，loga的居中问题，字母e的倾斜问题，还有一个是各字母的颜色与原版存在色差。居中问题用了height标签调整上边距，padding-left调整左边距，其中每个字母都看成一个块，所以padding-left不能定义在span全局。故只对第一个G进行了定义。颜色问题，截取了原版图片保存，用计算机自带画图软件打开，取色就可以得到颜色的具体参数，最终用RGB函数获取。关于最后‘e’的左倾问题，一开始只找到的倾斜函数，但只有右倾，后来使用了transform：rotate（），对e进行了22°逆时针旋转，达到了相似效果。
