4.Tips

  • win10自带的防火墙会组织windos与虚拟机的互相访问，在配置好后关闭防火墙。
  • windos和ubuntu都用DHCP自动分配IP，可以在不同工作环境适用。
  • 配置好后windos和buntu互相ping通表明连接成功。
  • 若ping超时，可能是网关的问题，在/etc/network/interfaces 文件中加入 nameserver 8.8.8.8 nameserver 8.8.4.4 ，修改完成后重启网络配置，sudo serverice networking restart。
  • 本次使用的是ubuntu desktop版本，存在GUI界面，占用内存，可以手动关闭。
    sudo service lightdm stop。
