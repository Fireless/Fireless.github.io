南京旅游

2018-09-18 Daily

南京

1 交通：

我完全信任高德，过去基本高铁，到了之后大部分景点都在地铁附近，所以不慌。PS：不过不能直接刷支付宝进站，要网上买票再进，略烦。

2 住宿：

倾向于民宿，有点想去看下南京大学的招待所。这个等到时候时间和行程定了再看，不然没法定。

3 景点

3.1 总统府——南京博物院一带

3.1.1 总统府

票价：据说美团只要19元
简介：热门景点，人多。主要是感受一下民国建筑风格和园林。附近有很多博物馆，六朝博物馆，江宁织造博物馆什么的，有个美术博物馆评价还蛮好的，某人可能有兴趣。还有对面是南京图书馆，亚洲第四大图书馆。
预计时间：3~4h

3.1.2 甘熙故居

票价：20元
简介：小众景点，安静。人文气息和历史气息比较重，有南京民间传统手工展示，不定时有现场制作。
预计时间：1h

3.1.3 梅园新村

简介：就是个小村子，政治正确。存放着我党的历史资料，有周恩来铜像和周恩来图书馆.
预计时间：大概给个一节课45min？30min是给图书馆的。

3.1.4 南京博物院

票价：免费，九点开馆。
简介：有六个分馆历史馆、特展馆、数字馆、艺术馆、非遗馆、民国馆。好像大部分人是去民国馆拍照的？有一条民国风格的老街。
预计时间：其实可以逛一天。

PS：

感觉大概南京这部分区域就是个古董铺子，还有很多什么寺啊，庙的，并没有什么兴趣。要去的话可以到时候顺路瞅一眼。

3.2 玄武湖一带

3.2.1 鸡鸣寺

票价：10元
简介：樱花路很出名，但现在并没有樱花。emmm、有兴趣的话可以去吃个素斋。
预计时间：我只是帮他当一个备选的吃饭的地方。

3.2.2 玄武湖

票价：免费
简介：想在南京感受一下大自然的气息，大概也就这里了，湖边走一走，坐个船什么的。栖霞山离车站比较远，暂时不考虑。
预计时间：：1~2h

3.2.3 台城城墙

简介：我把这个地方列进来只是因为他在城墙上可以同时看到鸡鸣寺和玄武湖。emmm，是不是直接去这里就完事了？
预计时间： 1h以下

3.2.4 狮子桥步行街

简介：看名字就知道，这又是一个，吃吃吃的地方。

3.2.5 南京站

玄武湖这一片区离南京站比较近，基本作为行程最后一个区域，逛完直接南京站上车。

3.3 some places i want to go

东南大学四牌楼校区—南京大学鼓楼校区—先锋书店五台山店

这条线比较有意思，可以顺路去个一个到两个地方。好了 懒得找图了。

3.4 还有一些单独的景点

3.4.1 line friend

拍照圣地？不是很了解。免费但人很多，多到排队。去的话得趁早。

3.4.2 江宁爱情隧道

大概也是个拍照的地方。

3.4.3 六和画家村

3.5 秦淮河一线

因为某人去过了，就不列进来了，不过南京南站下车，找住宿的话，离这边和很近。想去的话晚上可以出来稍微逛下。

3.6 吃

其实我对吃的要求没那么高，不过看到网上有个小哥每顿吃2~3种小吃，吃个两天南京特色小吃也吃的七七八八了，你们加油。

4 总结

大概就是南京南站下车，先去总统府一带，然后第二天慢慢往玄武湖靠，到南京站结束。中间列的景点我们两天肯定逛不完，可以挑一点喜欢的地方去一下，你们有什么想去的地方也可以再加上去。最后要去的地方定了，路线什么的稍微思考一下就好了。。emmm、还有南京地铁不准喝水，不准带食物，不知道真的假的。好了、目前就这么多。
