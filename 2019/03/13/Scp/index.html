通过ssh连接ubuntu虚拟机服务器及下载文件

2019-03-13 work

简介：安装完ubuntu server后，希望可以从windos下对其访问，以达到windos下工作，代码完成后部署于虚拟机linux系统的目的。主要使用了ssh连接，并利用scp命令进行文件的传输。

1.开启ssh服务

sudo ps -e |grep ssh

查询ssh服务是否开启。如果已开启ssh服务 会显示监听的端口，若ssh还未开启则先安装ssh。

sudo apt-get update
sudo apt-get install openssh-server
安装完成后重启ssh服务

sudo service ssh restart

2.查询虚拟机的IP

可以看到IP地址为192.168.1.5。

3.SSH连接ubuntu server

ssh 用户名@192.168.1.5

在windos下，通过ubuntu用户名和IP申请访问，输入密码后即可访问ubuntu server，此时即可在windos下对虚拟机的文件进行操作。

4.scp传输文件

可以看到虚拟机目录下有README.md文件，退出虚拟机后，在windos下利用scp命令就可以获取该文件。

scp stan@192.168.1.5:~/README.md ./

格式为 scp 用户名@ IP ：需求文件目录 文件存放目录

至此ubuntu server环境基本功能配置完毕。
