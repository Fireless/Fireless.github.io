Vue学习笔记（四）---API的调用

2019-04-03 vue

简介：

通过查看vue官方文档，尝试调用cnode的社区的api重构社区。

1.安装axios

通过yarn或者npm安装axios

2. 引用axios

在需要使用api的页面声明引用axios

3.使用axios发送请求

在生命周期函数mounted下 发送axios请求，根据cnode的api说明此处使用get请求，然后使用then函数获得返回的信息

注意此处返回的.data.data.具体要用什么参数需要F12进入浏览器调试界面的network下查看返回的数据样式。
