Vue学习笔记（三）--路由问题

2019-04-02 vue

简介：

在使用vue-router的过程中，发现点击跳转按钮路由无反应，经过研究发现是因为标签的问题

路由配置的主要步骤

1.首先需要引入vue-router，由于本项目是由vue-cli构建，在创建时已经选择了增加vue-router路由，所以此步骤省略。

2.在main.js文件中引用router，并声明使用

3.配置路由文件/router/index.js

此处需要注意先对相应组件进行注册,再进行路由的填写。

4.router-view

初次使用时路由一直无法正常渲染。发现是没有使用的原因。该标签作用是渲染匹配的的路由。我理解为该标签所在的位置就是SPA页面跟随路由变化的那一部分。所以我在入口文件App.vue的template中只加了导航栏和最终效果即为导航栏不变，而router-view部分随路由改变。
