Python学习笔记1：环境配置

2018-02-06 Python

Python简介

Python是目前使用频率最高的编程语言，其最大特点就是import，可以通过导入各种现成的库，实现各种所需功能，相比于Java可以大大提高项目完成的速度。（这里不讨论项目十分复杂，需要二三十人甚至更大规模团队开发的情况。就新人小白而言，Python确实更加友好）。只需要学习各种 相关的库，加以引用就可以实现编程。而作为一种开源的语言，目前Python库种类非常丰富，涵盖各行各业，Google的搜索引擎，YouTube上视频的爬取，目前大火的数据分析，人工智能都可以用到Python。

Python环境配置

1、版本选择

目前Python有2和3两个版本同时存在。虽然说Python 3已经基本趋于成熟，但还有不少企业、项目仍在使用Python 2。而Python官方虽然说过2020年将停止维护Python 2，但真实性尤未可知（之前他们就说过2014年全面下线Python 2，但之后发布公告称由于工作量巨大，暂缓下线，结果到现在Python 2还是很火，orz）。关于两个版本的优劣在此不做评价。选择自己需要的即可，两者语法上只有很小区别，适应另一个版本的时间不需要太久。我选择了Windows下的Python 3进行安装。

2、下载并安装

Python 的安装包可以直接去官网下载，支持Linux，mac OS，Windows，可根据自己的情况选择版本。

Python官网：https://www.python.org/

在安装时需要注意的一点，勾选add Python to PATH的选项，这样安装时会自动帮你配置环境变量。如果忘记勾选，需要自行配置环境变量。具体步骤为 ：打开我的电脑–属性–高级系统设置–环境变量–系统变量,在PATH条目最后自行添加Python的环境变量。其他设置全选默认直接下一步即可。

安装完成后可利用cmd查看是否配置成功，此处看到我的版本号为3.6.4

利用ssh将本地项目部署至github

2018-02-05 ssh

背景：准备利用hexo + Github搭建自己的个人博客，在本地项目基本完成后，需要将本地项目利用ssh部署至Github仓库。

一、ssh的获取及配置

1.首先需要hexo，git bash 等软件，具体下载安装步骤网上很多，此处不赘述。

2.利用git bash获取本地和公共密匙

获取命令为 ssh-keygen -t rsa -C "个人邮箱",结果如下：

注意此处会让你填写密匙生成地址和访问密码，若不设置，以后不用输密码直接访问，一路按enter即可。

3.查看密匙并提交至github

密匙文件夹的一般路径为~/.ssh

用文本编辑软件打开id_rsa.pub（我用的是notepad），复制全部内容。即为SSHkey。

打开github，在个人设置中将SSHkey添加至github账户。

4.检查链接是否成功建立

输入ssh -T git@github.com


返回如图结果说明ssh配置已完成。

二、_config.yml文件修改的注意事项

在hexo d的时候遇到了如下问题：

Please make sure you have the correct access rights and the repository exists.

首先确认sshkey的配置是否有误，如错误则删除sshkey，并重新配置，具体方法如本文第一节所述。如果配置真确，则大概率是_config.yml文件的问题。

本地的配置主要是修改博客根目录下的_config.yml文件，再文件最后加上远程仓库地址的信息。

这里列举几点需要注意的细节：

1、type类型为git ，而不是github。

2、type，repository，branch后都必须有一个空格，否则无法识别。

3、网上很多教程写仓库地址格式为https://github.com/***,https格式很多时候会出现错误，无法部署，这里我改成git@github.com.

确认_config.yml文件无误后，就可以通过hexo generate，hexo deploy进行部署。

三、关于CNAME文件的问题

hexo deploy时遇到每次部署CNAME文件都会被自动清除导致网页无法显示的问题。此时只要将CNAME文件放在source文件夹下再 hexo g 就可以避免这个问题。readme文件同理。
